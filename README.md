# EXPLOSIVE_COOKING
A prototype NES game combining "Overcooked" and "Duck Hunt"

## How to Play
This game requires both an NES compatible controller in port 1 and an NES Lightgun in port 2
Player 2 (Mheet) must defend the kitchen by shooting enemies with the Lightgun
Ammunition is limited and can only be crafted by Player 1 (Dheeg) cooking at various stations in the kitchen
